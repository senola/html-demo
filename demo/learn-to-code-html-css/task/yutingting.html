自由行走的花 喻婷婷

一处未见过的风景，一种没有听过的声音，去脚踏那座山，去追寻那段情，自由的只想放空自己……关于旅行，你或许有无数个出发的理由，但在现实的面前，理由变为行动，转换成一个背包，一台相机和一段跋涉，实在需要太大的勇气。

从17岁开始有了第一次旅行后，喻婷婷的行走就一直没有停息。工作之余约上几位好友，横跨草原和沙漠，穿越森林和雪山，丈量平地和高原，希望在自己年轻的时候，多走自己没走过的路，多看那些未看过的风景，在旅途中发现最真实的自己。

在去西藏的旅行中，婷婷担任临时团队的财务。她把团队经费全部记录到随手记中，一旦有集体花销，就记一笔。旅行结束后，把账本分享给所有成员，不仅让大家清楚知道西藏行的花销流水，更给大家留下了最美好的回忆。

说到旅行，还得从我17岁那一年说起。当时是暑假，我和爸妈有了人生中的一次外出旅行，目的地是西安。我知道那是一座文化底蕴非常深厚的城市，但是当课本中描写的兵马俑、阿房宫穿越过数千年的风霜来到我的眼前，深沉而又倔强地屹立在那里时，我还是被震撼到了。当年那些匠人们低沉的号子声，修建时所留下的血和汗，以及他们的苦心孤诣，似乎都发生在昨日，带着一连串关于战争、贫穷、流离、繁荣、兴衰的故事……这些感触，让我豁然开朗，从此我便爱上了旅行。

这些年来，只要一有时间，我就会背上背包，带上相机，约上几个驴友，开始出发。从草原到沙漠，从森林到雪山，从平地到高原，每一次旅行对我来讲都充满了新奇，或风土民情历史文化，或旖旎风光路途经历，这些都会促使我去思考，从而发现自己身边的美好和残缺，让我更懂得珍惜。

记得20岁的时候，我第一次踏足高原。当时有一个户外旅行团队为了开辟新的旅行路线，决定去雅拉雪山实地考察。雅拉雪山位于川西以南，在康定一带。早在2007年，我就曾经听说并远远地看见过这座雪山，向往已久。我一听这个消息，就毫不犹豫的决定一起跟着去。

没有经历过高海拔，说走就走，准备不充分……这给我们一路带来了非常多的困难。我们去的时候虽然已经是清明，但那时候的康定还很冷，入山过后，雪开始越下越大。我们赶紧找到向导出发，让马匹驮着行李，我们自己背着随身包，尽管如此，山路同样难行。到半山腰的时候，雪就很厚了。因为没有经验，做的饭也是生的，煮的肉也很硬以至没法下咽。加上出发匆忙，加厚衣服、防晒霜之类的物品都没有带，当时真的可以说是又冷又饿又累。

常有人说，无限风光在险峰。经过一天的徒步，我们终于登顶垭口，身处皑皑白雪，头顶触手可及的蓝天白云，不远处就是两颗晶莹剔透的高原湖泊。顿时，路上的疲劳、艰辛一下全部都忘记了，只管享受这人间天堂般的美景。

我想，这也是旅行的另一种意义吧。不管路途多么的艰难，只管勇往直前，因为我们知道总有一处风景在灿烂地绽放着等待我们的到来。所以有时候我会分不清是因为景美，还是因为沿途的经历而使景变得更美……

大自然的鬼斧神工自不限于此处，但是此次经历给我的印象最深。除了景美，更让我吸取了一些教训——高原上的紫外线非常强烈，而我因为准备不充分，仅仅在上面呆了一天，就被晒伤了。脸上起了很多小泡泡，而且整个都浮肿了起来，还蜕皮。下山后，我赶紧到医院去看，结果还是整整一年才恢复。现在晒太阳多了，也会有些疼痛。

我分享这个经验，其实就是想告诉大家，说走就走看起来很逍遥，但是没有充分的准备，对旅途的考察，还是不要轻易出发。生命和健康始终是第一位！包括现在有的女生选择单独出行，我也觉得这是一件非常危险的事，我们出发是为了美好，更要为自己和家人负责。

走出去虽好，最终还是要回到现实生活。从学校毕业出来，我换过几次工作，现在是重庆晚报的一名记者。不管岗位、行业如何变化，我始终如此地热爱它们——因为我知道，工作为我带来的不仅仅是物质的满足，让我有足够的资本再次出发，更是我社会价值的体现。我认为只有这两者统一了，才算是一个真实而饱满，脚踏实地而心怀梦想的的人。

因为经常和朋友结伴远行，所以如何处理临时团队的财务是一个很大的问题。在朋友的推荐下，我选择了随手记。记得上次去西藏，是由我来保管团队的资金，我就在随手记中先新建了一个旅游账本，把大家给我的旅行经费记录进去，一旦有集体花销，就记一笔。旅行结束后，直接把账本分享给团队成员，不仅让大家清楚地知道了我们本次西藏行各种花销流水，一笔笔的记录和备注，更给我们留下了最美好的回忆。

上路自有收获。以后一旦能够安排出时间，我一定会继续出发，希望趁年轻多走几步路，多欣赏一些沿途的风景，不因要急于抵达目的地而错过了流年里温暖的人和物；希望趁年轻多做些想做的事，不因瞻前顾后而停滞不前。接下来，我也会到像新疆、内蒙这样的一些宗教气息或民族文化浓郁的地方去，更多地感受在这片神奇的土地上那些我未知的文化和人生。
