海归商悠然做奢侈毛巾

创业，旁观者看起来很浪漫，路上的人坚持得挺忧伤，很多人怀揣着暴富的发财梦加入到滚滚盲流，倾囊而出后却还不知道自己出发的初衷。面对不到3%的成功率，这场突出重围的沼泽之战里，注定是智慧、勇气、毅力并举的虐己游戏。能让自己在接踵不断的挫折中依然坚挺的理由，那就是曾经打动过自己的灵感，和一个成长为更好的自己的梦想。

出国留学七年多，在金融行业的繁荣期里，商悠然曾就职于国际知名投资机构，因为一份对家庭温暖的眷念毅然决定回国，选择做最优质的毛巾产品传递自己所认为的那份温暖。以前花钱如流水，现在自己做经营方方面面都要精打细算，控制成本，稳健运营。

一年多来，悠然一直用随手记管理着自己的部分理财投资和店铺经营流水，店铺的管理都交给团队分工打理，自己的主要精力放在国内国外的主要行业展会上。无论何时，在全球各地，打开随手记就能查看同事每天准时更新的店铺交易流水，悠然最喜欢的就是这种便捷。

就我个人的成长经历来看，三分之一的时间在家乡，三分之一的时间在深圳，另外三分之一的时间在国外。父母忙于生意，我自幼在性格上就比较独立，14岁开始随父母参加广交会，经商意识启蒙的比较早。

从高中开始，我到澳洲悉尼求学，并一直读到大学毕业。对比中西方的教育，最大的感受就是对个性的尊重，上大学时，手工也可以成为一门课程，一切自主，老师所有的讲义都放在网上，可以自由选科。

国外的大学教育也比较注重实战，比如金融类一个学期要做一篇课程论文与考试，很多论文要团队来完成，通常是8-10人一组。在这个过程中可以充分的学习如何进行团队协调应变处事。在课程结业时，8个人当中经常只有3-4人能过关，团队最弱的人决定了团队的成绩。这种强调团队最弱的环节，让大家更紧密的合作在一起。期末考试如果不上心，是很难通过的。注重团队合作的教育让我在后期的创业中也相当注重团队的合作。

在澳洲获得比较大锻炼的还有生活自理能力，大学里只提供覆盖约20—30%学生的公寓，大部分学生的住宿要靠自理。我18岁开始在悉尼和同学自己租房子住，自己独立打理生活的一切。比如经常逛宜家，一二个人就能动手组装出一屋子的各式家具。培养出的独立动手能力对我以后的生活也起到了很大的帮助。

大学毕业后，我凭借读书期间不错的履历很顺利的进入到了一家国际知名的投行工作，一做就是4年。在这段金融行业的全盛时期，仿佛做什么都能成功，看准的项目都有不错的回报。辉煌的业绩下，员工的福利待遇都非常好，也做了很多不接地气事情。到亚太各地谈项目经常坐私人飞机，周末游艇party，宴请出入的都是各地顶级会所。亚太区的年会，一般都是集体到巴厘岛开。

08年金融危机后，整个行业从高峰一下子跌到谷底，身边的很多同事从年薪几百万到被裁员成为了常态。有的同事开始去印尼、泰国养生悟道，也有部分同事在香港、澳洲开起了咖啡店。有一次去美国旅行在朋友家住，看着她的爸妈在身边无微不至的照顾，一种家的温馨感油然而生，结束旅行后我就办理了辞职手续回到父母身边。

遇到我的先生并选择婚姻是我迄今为止作出的最重要的决定，也迎来了人生新的起点。怀孕期间，我想买些高品质的毛巾用品，然而逛遍了各类超市居然难得买到一条埃及棉毛巾。一些二流产品充斥着货柜，卖着令人咂舌的价格。这也反应出了一个现状，国内市场还不太注重毛巾消费的品质，高品质毛巾市场在国内基本是空白。

自己家族企业做的埃及棉毛巾产品全部用于出口到日本、欧洲、美国等地，自己何尝不从电商渠道开始做一个面向国内市场的高品质毛巾品牌来填补这个市场空白呢？老公也比较认可我的这个想法，怀孕期间我依然保持着行动派的作风，很快确立了产品规格，样式，搭建电商团队，生完小孩后，我将主要精力投入到了自有品牌的运营上。

“亦居YICASA”这个品牌名我酝酿了很久，寓意着用上亦居的毛巾，不管到哪里，独特专属的温暖体验，都将让你感觉家一样的舒适。在国内做高端毛巾产品首先要面对的就是市场教育的问题。

绝大多数的消费者都不知道，埃及长绒棉是目前全球最好的棉材料。因产量有限，埃及政府以配额的形式有选择的对外出口，并采用埃及亚历山大条款，以“埃及棉花仲裁、检测总局”出具的证书作为最终结算依据。顶级的埃及棉既有丝一样的质感，又不容易皱，洗水100次仍不变形，上面有光泽，近肤的感觉非常好。

埃及棉产品主要供应欧美五星级酒店，并以毛巾类奢侈品形式在零售渠道销售。家族旗下的公司是国内为数不多的能获得埃及棉进口配额的公司，并在毛巾出口贸易中位居前五位。产品品质和工艺拥有绝对的竞争力，这点我很有信心，店铺销售到现在用户口碑一直非常好。

很多人做电商平台喜欢做加法，找更多的货源，拼性价比，刷爆款走量。我在经营策略上做的是减法，亦居YICASA只主打两类产品：一是自产的埃及棉毛巾，二是有机棉用品。我在前期并不急着冲量，先从懂我们产品的人开始做口碑，在有消费能力的人群中培养注重毛巾产品的消费文化。我的电商平台运营一直坚持两点：不打折，不冲销量。我希望再过2-3年能成功激活国内高端毛巾市场。

我是一个非常注重家庭的人，做为一个准妈妈面对事业、家庭和孩子，我期望的是一种比较完美的平衡。在小孩的教育上我的方式比较西化，一直坚持粗养的方式，不像国内的父母过度溺爱与呵护。在冬天我不会给小孩穿过厚的衣服，培养她的自然抵抗力。在家里做好防护措施后，我会让Baby自由的玩耍，满地爬。

我坚持让Baby自己吃饭，他自己吃一次饭，桌上和地板上肯定都是汤汤水水，我要多做两个小时的善后工作，但是这些过程是他自己探索人生的重要启蒙阶段不能省略。

国外的家庭，一般没有保姆，没有父母老人会给子女带小孩，小孩一岁就要有独立吃饭的能力，培养出独立的个性与习惯非常重要。我的Baby十个月就自己学会了走路。他生病了以后基本上不去医院，发烧后都在家里自己处理。我在澳洲十一年没有打过针，经常性的静脉注射是国内特有的现象，这样对小孩的免疫力是一种破坏。

由于经常要在国内国外各地跑，电商平台的经营暂时还不需要专业的企业级财务软件，身边的闺蜜给我推荐了随手记。每天下午6点都能准时收到当班主管同步过来的店铺经营流水，无论在国外或者是外地，通过随手记我都能对店铺的经营情况了如指掌，这点非常方便。有时需要为宝宝做大采购时我会记一下宝宝账。与闺蜜们约定出国血拼时我会记录下大额花销适当控制下预算，对财务较为敏感可能和我学金融出身有较大关系。

未来，我希望通过几年的努力，实现电商平台年销售额过亿，成为中国高端毛巾市场的开拓者。但是在规模上我还是会坚持小而美，生活充实即可。就像我之前说的一样，对于事业、家庭和生活，女人最理想的状态就是做到和谐平衡。
