传媒黑马李翔

每当生活面临重大变化时，弄清自己的财务状况是做出适当抉择的首要因素。还剩多少钱？该为家人买份保险吗？买什么类型的理财产品合适？朋友的项目适合投吗？现金还能撑多久？负债是否过高？……如何一个人需要持续记账，系统管理自己的个人财务，意味着他的人生已经进阶了。至今为止，已有超过6000万人在使用随手记，每天使用随手记记账的人超过300万。每一笔账对应的是生活的改变，是成长轨迹，是拼搏的人生。本期随手记人物周刊，我们带你解码北方的性情汉子，中娱传媒董事长李翔。

作为拥有四家分公司的传媒集团老板，CIP国际超级模特大赛中国区执行主席，电影投资人，李翔的沉稳和冷峻让人忽略了他还是一个80后。坐在商务谈判桌上，很多人忍不住问起他的背景，更多人认为他是一个富二代。对此李翔坦言，我没有特别的出身，来自北方普通的家庭，这些年丰富的经历，练就了一个泰然心态。大家关心我的背景更多是因为我年轻。

李翔上大学时就特爱折腾，在学校开过书店、洗衣店、花道炒饭馆，被同学戏称为“李院长”。从大二开始，李翔没有再向家里要过一分钱，每个月还能赚三千多元。一次偶然的机会，李翔应聘到一家传媒公司，上班很刻苦带点儿拼命，起早摸黑的干。4个月就从业务员做到了副总经理。到了大四当其他同学焦急的四处找工作时，他已经以副总的身份回到母校招聘人才。

06年，觉得羽翼丰满的李翔回老家开了自己的第一家传媒公司。创业起步同样艰辛，开业当天李翔一边忙着庆典，一边要应付上门要钱的物料供应商，公司就这样踉踉跄跄的上路了。通过2年辛苦的打拼业务开始稳定，并在哈尔滨、济南、烟台、秦皇岛陆续设立了分公司。

08年，李翔开始投资拍摄电影。先后投资了电影《惶惑》、《残冰》、《指尖的寂寞》，最新筹拍《心愿》和《功夫小子》已经开机。刚涉足电影投资时曾给李翔带来过一次不小的危机。电影拍到一半预算就花完了，本来公司的资金链就比较紧张，他最后将房和车卖掉凑齐资金硬顶着将电影拍完。说到这些年投资电影，李翔唯一的遗憾就是还没有一个成名作。他目前也正在策划投资制作让大家耳目一新的影视作品。

为了让公司更上一个台阶，李翔将公司总部搬到深圳，一方面因为深圳这个城市和他一样年轻有活力，另一方面毗邻香港更方便对接香港的演艺资源。从英国拿下了CIP国际职业认证的中国代理权后，李翔倾尽全力投资这个项目，目的是打造中国艺术类的ISO9000认证.三年来，中娱传媒的CIP项目已经拥有1000余家客户，并且每天都在快速增长。按照目前良好的发展趋势，CIP国际认证成为中国艺术类的第一大国际认证已经是指日可待的事情。

“来到深圳后我开始使用随手记，因为工作太忙，项目多，资金流转快。有时候上飞机前接到电话，下飞机就要做出投资的决定。在来不及看财务报表的情况下，用随手记看一下大致的财务状况利于我及时决策。项目记账在这方面挺便利的，我现在改用多账本功能了，一个投资项目配一个账本。”李翔表示。

目前，中娱传媒核心业务集中在承办全国性的选秀比赛，电影投资， CIP国际认证，大型活动策划，并在国内形成了自己的独特优势。“我愿意做传媒界的一个基石，通过中娱传媒提供的平台，让更多有梦想的孩子走向舞台。未来我希望将中娱传媒打造成为中小型传媒公司的孵化器和成长平台，帮助更多有梦想的传媒公司快速走上正轨，获得成功。”李翔对中娱传媒的定位很清晰。
