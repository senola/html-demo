“display” 属性

display 是CSS中最重要的用于控制布局的属性。每个元素都有一个默认的display值，这与元素的类型有关。对于大多数元素它们的默认值通常是block 或inline。一个block元素通常被叫做块级元素。一个 inline 元素通常被叫做行内元素。
